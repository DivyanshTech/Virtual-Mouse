# Virtual Mouse using Computer Vision 🖱️

A real-time **Virtual Mouse system** built using **Python, OpenCV, MediaPipe, and PyAutoGUI** that allows users to control the computer cursor using hand gestures through a webcam.

This project demonstrates **Computer Vision, Hand Tracking, and Human-Computer Interaction** concepts in a practical implementation.

---

## 🚀 Features

* Real-time hand tracking using MediaPipe
* Control mouse cursor using index finger
* Smooth cursor movement
* Click action using thumb-index finger gesture
* Live webcam interface
* Automatic demo video recording
* Lightweight and fast execution

---

## ⚙️ Installation

### 1. Clone the repository

```
git clone https://github.com/DivyanshTech/Virtual-Mouse.git
cd Virtual-Mouse
```

### 2. Install dependencies

```
pip install opencv-python mediapipe pyautogui numpy
```

### 3. Run the project

```
python virtual_mouse.py
```

---

## 🎮 How it Works

1. Webcam captures hand movements
2. MediaPipe detects hand landmarks
3. Index finger controls mouse movement
4. Thumb and index finger distance detects click
5. Cursor moves smoothly on screen
6. Demo video automatically records in `.avi` format

Press **Q** to exit.
